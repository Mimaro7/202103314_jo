-"vi"sual editor
  -vim -vi improved
-프롬프트에서 파일을 직접 편집할수 있게 해주는 툴
-특징
  - 화면 단위 편집기
  - 유닉스 계열에서 가장 많이 사용됨
  - 텍스트 편집 기능만 제공함
  - "less"
  - 모드 : 일반모드 <--> 입력모드 <-->명령모드
    -ESC로 모드 전환
    -명령모드 : 파일 저장, 덮어쓸건지, 빠져나갈건지 등등
    => vi로 파일 열기 -> 입력모드 -> ESC -> 빠져나가기
    
    -입력모드:
      -i:현 커서에 입력
      -a:현 커서 한칸 뒤에서 입력
      -o:한줄 추가 후 입력
    ex)vi example 후에 i 눌러봄

    -명령모드
      :wq! + enter
    ex) 파일 아무거나 열어서 수정하고 저장하고 나오기
    
    set -o vi =>안대는 사람은 한번쳐보기 또는
